GET /

  Returns the list of URLs that can be requested.
  Response format: JSON.
  Parameters: none.



POST /clients/new
  
  Creates a new client. Returns the client.
  Response Format: JSON.
  Parameters:
    name        Client's name and last name
    user        Client's user
    password    Client's password

  Example request:
    POST http://example.com/clients/new
    name="Juan Perez"
    user="jperez"
    password="p4$$w0Rd"

  Example result:
    {
      "id" : 42,
      "name" : "Juan Perez",
      "user" : "jperez",
      "creditCards" : null
    }



POST /clients/login

  Authenticates the client. Returns the client.
  Response Format: JSON.
  Parameters:
    user        Client's user
    password    Client's password

  Example request:
    POST http://example.com/clients/login
    user="jperez"
    password="p4$$w0Rd"

  Example result:
    {
      "id" : 42,
      "name" : "Juan Perez",
      "user" : "jperez",
      "creditCards" : null
    }



POST /clients/add-creditcard
  
  Adds a credit card to the authenticated client. Returns client.

  post number, security_code and description; returns client

  Response Format: JSON.
  Parameters:
    number           Credit card's number
    security_code    Credit card's security code
    description      Credit card's company

  Example request:
    POST http://example.com/clients/add-creditcard
    number="1234567890123456"
    security_code="123"
    description="visa"

  Example result:
    {
      "id" : 42,
      "name" : "Juan Perez",
      "user" : "jperez",
      "creditCards" : [ {
        "id" : 1,
        "number" : "1234567890123456",
        "securityCode" : "123",
        "description" : "visa"
      } ]
    }

GET /shopping-cart

  Returns the shopping cart for the authenticated client.
  Response Format: JSON.
  Parameters: none.

  Example request:
    GET http://example.com/shopping-cart

  Example result:
    {
      "id" : 3,
      "itemLines" : [ ],
      "total" : 0.0
    }



POST /shopping-cart/add-product

  Adds products to the authenticated client's shopping cart. Returns shopping cart.
  Response Format: JSON.
  Parameters:
    product_id       Product's id
    quantity         Product's quantity

  Example request:
    POST http://example.com/shopping-cart/add-products
    product_id="3"
    quantity="4"

  Example result:
    {
      "id" : 3,
      "itemLines" : [ {
        "id" : 4,
        "product" : {
          "id" : 3,
          "description" : "coke",
          "price" : 5.0,
          "category" : {
            "id" : 2,
            "description" : "drink"
          }
        },
        "quantity" : 4,
        "subTotal" : 20.0
      } ],
      "total" : 20.0
    }
